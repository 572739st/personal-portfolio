# Personal Portfolio — S1 · S2 · S3

A responsive, interactive portfolio showcasing three semesters of things I worked on during my study at Fontys ICT.

This project brings together 9 projects across 3 semesters into one structured portfolio experience.

---

## Live Demo



---

## Overview

This portfolio was built to:

* Present my development across three semesters
* Combine technical and creative work in one structured system
* Make projects easy to explore and navigate
* Support internship and job applications with clear project context

Each project includes context, role description, tools used, and supporting media such as videos, images, and presentations.

---

## Features

* Semester-based structure (S1, S2, S3)
* Skill-based project filtering
* Expandable project descriptions
* Video modal system for project media
* Image presentation lightbox
* Smooth scrolling navigation between sections
* Responsive layout for desktop and mobile (not fully mobile-optimized)
* Custom animations and transitions

---

## Key Functionality

### Skill Filtering

Projects can be filtered by skills such as JavaScript, Branding, Figma, and Filmmaking. Clicking a skill tag highlights relevant projects and dims non-matching ones.

### Expandable Project Details

Each project includes a “Read more” section with additional information about process, challenges, and individual contributions.

### Media System

* Video modal for project videos
* Image lightbox for presentation slides
* External links for live demos and YouTube content

### Semester Navigation

Users can jump directly to Semester 1, 2, or 3 using the interactive navigation system.
